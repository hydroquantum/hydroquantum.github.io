# HydroQuantum.github.io

This site is live on [https://hydroquantum.github.io](https://hydroquantum.github.io)

HydroQuantum is a platform dedicated to advancing quantum computing in hydrology and water resources. It explores quantum algorithms and quantum-enhanced AI to improve modeling, prediction, optimization, and uncertainty analysis for complex water systems.

This repository now contains a website for the quantum computing branch of Clemson University’s Hydrosystem and Hydroinformatics Research (HHR) Lab.
